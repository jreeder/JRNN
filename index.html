This is a Neural Networks library in C++.

It makes extensive use of boost shared pointers and some containers, so Boost version 1.43 or greater is needed to compile against.

The experimental programs make use of TCLAP for command line parsing, and xmlconfig/tinyxml for parameter file reading.

xmlconfig is an extension of tinyxml that provides a very limited XPath like interface into xml files. It has built in value cacheing, and maintains awareness of the xml file state, so look ups are quick and values are up to date with the original xml file.

Network serialization is implemented in a way that allows for several different output formats. Currently the only output format is JSON which is implemented using the JSON-Spirit Library.

Currently there are trainers for Standard Backprop, RProp, and Cascade Correlation.

This software is of primary use to my disseratation research. So things are pretty specific to my needs. Once I've finished with my dissertation I'll clean this up and make it more presentable.

Dependencies & Use

  • Boost Version >= 1.43
  • TCLAP Version >= 1.2.1
  • JSON-Spirit Version >= 4.05

Currently only the VS2010 build environment is kept up to date. At some point I will be adding cmake support.
